**Project4_Resume_Ranking_System** contains the codebook where a resume ranker has been built on Python which processes both Text and PDF formats of resumes along with a Job Description to give a ranking based on person-job fit.


**app.py** and **requirements.txt** were coded to create the front-end website on Streamlit. The link to the website: https://resumerankingsystem-k401.streamlit.app
